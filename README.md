# Project Overview

This repository contains the core systems, tools, and examples used in the project.  
It is organized into several main sections for easier navigation and understanding.

---

## 1. UI

This section covers the UI architecture and implementation:

- MVVM-based structure with ViewModels, UI binders, and models
- Core UI services
- UI animations
- Real usage examples

---

## 2. Tools

This section contains editor tools and utilities:

- Examples of Unity editor extensions
- Dropdowns, windows, and custom editor elements
- Editor scripts for build preparation

---

## 3. System

Divided into two parts:

- **LocalNotification** – a simple, isolated feature demonstrating how to build reusable functionality
- **Analytic** – a modular system that can be reused across multiple projects

---

## 4. General

This section includes miscellaneous examples and utilities that support development of other systems.

---

## 5. Gameplay

This section demonstrates a compositional (component-based) approach to character design:

- Component-based character setup
- FSM for character state management
- Command system examples
